🗂️ Case Management

Facility users can create, locate, and manage death cases. Ownership and accuracy are critical for compliance and downstream reporting.

➕ Create New Case

  • Use the Create Case button from the dashboard
  • Enter decedent details and verify against existing records
  • System will flag potential duplicates—review before proceeding

🔄 Ownership Actions

  • Claim: Take ownership of an unassigned case
  • Transfer: Send case to another facility or role
  • Relinquish: Release ownership if no longer responsible
  • Abandon: Request to void a case with justification

📜 Case History

Each case includes a full audit trail showing status changes, user actions, and timestamps. This supports accountability and legal compliance.
