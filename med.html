# chante-healthcare-training

🏥 eVitals Facility Administrator Training

This module supports licensed healthcare facilities in understanding their responsibilities within Pennsylvania’s eVitals system. Designed for adult learners, it blends clarity, accessibility, and practical application.

🎯 Objectives

  • Define Facility and Deputy Administrator roles
  • Manage user profiles and access requests
  • Verify training completion and documentation
  • Navigate technical setup and browser plug-ins
  • Register via Keystone ID and access eVitals

📚 Module Structure

  1. Welcome & Logistics
  2. Role Definitions
  3. User Management Responsibilities
  4. Training & Certification Requirements
  5. Technical Setup
  6. Business Partner Registration
  7. Resources & Support
  8. Wrap-Up & Q&A

📎 Supplementary Materials

  • Training Manual
  • Plug-in Installation Guide
  • Keystone ID Instructions
  • FAQs for Facility Administrators

🧠 Instructional Design Notes

  • Plain language and visual cues for clarity
  • Interactive elements and chat prompts
  • Accessibility: alt text, keyboard navigation, downloadable formats
